# AverqonTube

A high-density, cinematic streaming platform dashboard powered by the YouTube Data API.

## 🚀 1-Click Deployment
To instantly deploy this dashboard to your own Vercel account without the link breaking or returning a 404 Error, click the button below:

[![Deploy with Vercel](https://vercel.com/button)](https://vercel.com/new/clone?repository-url=https://github.com/MuneeswaranMD/averqontube&env=VITE_YOUTUBE_API_KEY,VITE_YOUTUBE_CHANNEL_ID,VITE_YOUTUBE_STUDENT_CHANNEL_ID)

> **Note:** When deploying, Vercel will pause and ask you for the 3 Environment Variables listed above. Make sure to paste your genuine `AIzaSy...` YouTube API key, not a channel ID!
